1. Clone this repo. <br>
2. cd finn  <br>
3. cd backend <br>
4. npm i  <br>
5. npm run start  // to start the backend servers. <br>
6. cd .. <br>
7. cd frontend <br>
8. npm i  <br>
9. npm run start
